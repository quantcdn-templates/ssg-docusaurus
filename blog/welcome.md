## Getting Started with Blogging

Docusaurus creates a **page for each blog post**, but also a **blog index page**, a **tag system**, an **RSS feed**...

## Creating New Posts

Create new blog posts by adding Markdown files to the `blog` directory. The filename format is:

```
YYYY-MM-DD-your-post-title.md
```

## Blog Features

- **Front matter** - Add metadata like title, authors, and tags
- **Truncation** - Use `<!-- truncate -->` to create a summary
- **MDX support** - Use React components in your posts
- **RSS feed** - Automatically generated at `/blog/rss.xml`

## Learn More

Check out the [Docusaurus blog documentation](https://docusaurus.io/docs/blog) for more details.
